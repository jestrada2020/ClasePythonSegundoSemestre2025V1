Paso 1: Estudiar el Concepto

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum en 1991 y se caracteriza por su sintaxis clara y legible, lo que lo convierte en el lenguaje ideal para principiantes y expertos.

🎯 ¿Por qué Python es perfecto para ti?
  • Fácil de aprender: Sintaxis similar al inglés natural
  • Versátil: Desde web hasta IA, todo es posible
  • Comunidad activa: Millones de desarrolladores te ayudan
  • Oportunidades laborales: Alta demanda en el mercado
  • Futuro asegurado: Crecimiento constante en popularidad

Características Principales:

  • Interpretado: No necesita compilación previa
  • Multiplataforma: Funciona en Windows, Mac, Linux
  • Tipado dinámico: No necesitas declarar tipos de variables
  • Orientado a objetos: Soporta programación orientada a objetos
  • Extensible: Miles de librerías disponibles

Sintaxis Básica:

# Comentarios en Python empiezan con # print("¡Hola, mundo!") # Esta es la función print # Variables (no necesitas declarar el tipo) nombre = "Python" version = 3.9 es_genial = True # Operaciones básicas suma = 5 + 3 resta = 10 - 4 multiplicacion = 6 * 7 division = 15 / 3 potencia = 2 ** 3 # Mostrar resultados print(f"Lenguaje: {nombre}") print(f"Versión: {version}") print(f"Es genial: {es_genial}") print(f"Suma: {suma}")

¿Qué son las Estructuras de Datos?

Las estructuras de datos son formas organizadas de almacenar y acceder a múltiples valores. Python ofrece estructuras integradas muy poderosas: listas, tuplas, diccionarios y sets.

  • • Métodos y operaciones de cada estructura

Paso 1: Listas en Python

Listas - Colecciones Ordenadas

Las listas son colecciones ordenadas y modificables que pueden contener elementos de diferentes tipos. Se definen con corchetes [].

# Crear listas frutas = ["manzana", "banana", "cereza"] numeros = [1, 2, 3, 4, 5] mixta = ["Python", 3.9, True, [1, 2, 3]] # Acceder a elementos (indexación) print(frutas[0]) # "manzana" print(frutas[-1]) # "cereza" (último elemento) print(numeros[1:4]) # [2, 3, 4] (slicing) # Modificar elementos frutas[1] = "naranja" print(frutas) # ["manzana", "naranja", "cereza"] # Métodos útiles frutas.append("uva") # Agregar al final frutas.insert(1, "pera") # Insertar en posición específica frutas.remove("naranja") # Eliminar por valor elemento = frutas.pop() # Eliminar y retornar último print(len(frutas)) # Longitud de la lista

Paso 2: Diccionarios en Python

Diccionarios - Pares Clave-Valor

Los diccionarios almacenan datos como pares clave-valor. Son muy eficientes para búsquedas rápidas y se definen con llaves {}.

# Crear diccionarios estudiante = { "nombre": "Ana", "edad": 22, "carrera": "Ingeniería", "activo": True } # Acceder a valores print(estudiante["nombre"]) # "Ana" print(estudiante.get("edad")) # 22 print(estudiante.get("telefono", "No disponible")) # Valor por defecto # Modificar y agregar estudiante["edad"] = 23 # Modificar estudiante["telefono"] = "123-4567" # Agregar nuevo # Métodos útiles print(estudiante.keys()) # Obtener todas las claves print(estudiante.values()) # Obtener todos los valores print(estudiante.items()) # Obtener pares clave-valor # Eliminar elementos del estudiante["telefono"] # Eliminar clave específica valor = estudiante.pop("activo") # Eliminar y retornar valor

Paso 3: Tuplas y Sets

Tuplas - Inmutables y Ordenadas

Las tuplas son similares a las listas pero inmutables (no se pueden modificar). Se definen con paréntesis ().

# Crear tuplas coordenadas = (10, 20) colores = ("rojo", "verde", "azul") punto_3d = (1, 2, 3) # Acceder a elementos print(coordenadas[0]) # 10 print(colores[-1]) # "azul" # Desempaquetado x, y = coordenadas print(f"X: {x}, Y: {y}") # Métodos útiles print(colores.count("rojo")) # Contar ocurrencias print(colores.index("verde")) # Encontrar índice

Sets - Sin Duplicados

Los sets son colecciones no ordenadas que no permiten elementos duplicados. Se definen con llaves {} o set().

# Crear sets numeros = {1, 2, 3, 4, 5} letras = set("python") # {'p', 'y', 't', 'h', 'o', 'n'} # Agregar y eliminar numeros.add(6) # Agregar elemento numeros.remove(3) # Eliminar (error si no existe) numeros.discard(10) # Eliminar (no error si no existe) # Operaciones matemáticas set1 = {1, 2, 3, 4} set2 = {3, 4, 5, 6} print(set1 | set2) # Unión: {1, 2, 3, 4, 5, 6} print(set1 & set2) # Intersección: {3, 4} print(set1 - set2) # Diferencia: {1, 2}

Código de ejemplo para el proyecto:

# Importar librerías
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
# Crear datos de ejemplo
datos = {
'nombres': ['Ana', 'Juan', 'María', 'Carlos', 'Sofía'],
'edades': [25, 30, 22, 35, 28],
'salarios': [45000, 55000, 40000, 65000, 50000]
}
# Crear DataFrame
df = pd.DataFrame(datos)
print(df)
# Estadísticas básicas
print(f"Edad promedio: {df['edades'].mean()}")
print(f"Salario promedio: {df['salarios'].mean()}")
# Crear gráfico
plt.figure(figsize=(10, 6))
plt.bar(df['nombres'], df['salarios'])
plt.title('Salarios por Persona')
plt.xlabel('Nombres')
plt.ylabel('Salarios')
plt.show()
